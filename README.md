# 📘 Smart Reading Tracker (Firefox Extension)

Smart Reading Tracker enhances your browsing experience by visually highlighting headings and text content, helping you focus while reading. It provides non-intrusive visual cues without modifying page structure.

---

## 🚀 Features

* Highlight headings on hover for better readability
* Subtle highlighting for paragraphs and list items
* Toggle extension ON/OFF from popup
* Lightweight and non-intrusive
* Works across all websites

## 🛠️ Installation (Firefox)

### Method 1: Install from file (Recommended)

1. Open Firefox
2. Go to:

   ```
   about:addons
   ```
3. Click the ⚙️ (gear icon)
4. Select **"Install Add-on From File..."**
5. Choose the downloaded `.xpi` file
6. Click **Add**

---

### Method 2: Drag & Drop

1. Download the `.xpi` file
2. Drag it into an open Firefox window
3. Click **Add**

---

## ⚙️ Usage

* Click the extension icon in the toolbar
* Toggle ON/OFF as needed
* Hover over text and headings to see highlights

## 🧑‍💻 Developer Info

* Built using WebExtensions API
